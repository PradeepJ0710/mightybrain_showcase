# MightBrain

### Self-Development Mobile Application

A comprehensive personal development platform designed to enhance individual growth, promoting overall well-being.

This application includes 7 major modules - Learning, Daily Learning, Habits, Emotion Journal, Meditation, Reflex and Eagle Eye.

Each module targets different personal attributes such as memory retention, habit building, and mental health.
